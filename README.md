# react-raphael-chart

so easy, create charts with react-raphael!

## Develop

1. clone the repository

        git clone https://github.com/liuhong1happy/react-raphael-chart.git
    
2. install dependency package;

        npm install 

3. run example

        npm run example
        
4. build

        npm run build
        
## Use

Now , you can see [example](example/index.js) !
